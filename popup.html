Transcription

Native Messaging wakes the local host only when a recording needs transcription. Install it once with install-native-host.ps1; you do not need to keep a bridge window open.

Successful jobs also try to create an Anki card in the deck name you choose, with the audio on the front and the transcript on the back. If the deck name does not exist in Anki, card creation fails but the recording and transcript still save locally.

A strong starting point is ggml-large-v3.bin for better accuracy, but it is noticeably heavier and slower than smaller models on CPU.

Translation

When enabled, the extension opens the selected translation site in a background tab, pastes the transcript, waits for the translated text, then closes the tab and uses the result in Anki when available.

If the provider cannot be read or permission is not granted, the extension falls back to the transcript-only Anki behavior.
