          Native Messaging wakes the local host only when a recording needs
          transcription. Install it once with
          install-native-host.ps1; you do not need to keep a bridge
          window open.
        

        

          Successful jobs also try to create an Anki card in the deck you choose,
          using the field mapping below. If the deck does not exist in Anki, card
          creation fails but the recording and transcript still save locally.
        

        

          A strong starting point is ggml-large-v3.bin for better
          accuracy, but it is noticeably heavier and slower than smaller models
          on CPU.

Translation


        

          
          
        


        

          
          
        


        

          
          

            
            
          

          

            With a key, DeepL is translated over its API instead of its website:
            no background tab, and it keeps working while the browser is
            minimized.
          

        


        

          
        


        

          When enabled, the extension loads the selected translation site in a
          throwaway background tab with the transcript already in the URL, reads
          the translated text, closes the tab, and uses the result in Anki when
          available.
        

        

          Chrome throttles hidden tabs, so website-driven translation is
          best-effort and works best with the browser window open. A DeepL API
          key avoids the website entirely. If a provider cannot be read, the
          extension falls back to transcript-only Anki notes.
